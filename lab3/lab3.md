## Задание 3.1: Найти среднеквадратичное отклонение продаж.

Скрипт для выволнения запроса:

```sql 
SELECT 
    STDDEV(sales_amount) as sales_stddev
FROM sales;
```

## Задание 3.2: Сгруппировать продукты по первой букве типа, найти среднюю цену.

Скрипт для выволнения запроса:

```sql 
SELECT 
    LEFT(product_type, 1) as type_first_letter, 
    AVG(base_msrp) as average_price
FROM products
GROUP BY LEFT(product_type, 1);
```

## Задание 3.3: Вывести штаты, где разница между max и min продажей > 1000.

Скрипт для выволнения запроса:

```sql 
SELECT 
    d.state
FROM sales s
JOIN dealerships d ON s.dealership_id = d.dealership_id
GROUP BY d.state
HAVING (MAX(s.sales_amount) - MIN(s.sales_amount)) > 1000;
```
